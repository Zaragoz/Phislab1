                         Механика.
Механика - это раздед физики изучающий механическое движение.
Мы постоянно встречаемся с движением тел в повседневной жизни, в технической науке. Мы наблюдаеи движения людей, животных, воды, воздуха. В мировом пространстве движутся планеты, котметы, спутники. Движутся молекулы, атомы, электроны и другие элементарные частицы.
Механическое движение - изменение положения тела в пространстве относительно других тел с течением времени.
Основная задача механики - указать положение тела в определенный момент времени.

                         Кинематика.

                         Основные понятия кинематики.
Кинематика - это раздел механики, который изучает движение тела, не анализируя его причин.
Движение каждого тела можно рассматривать по отнорщнеию к любым другим телам. По отнощенмю к разным телам данное тело будет совершать различные движения: чемодан, лежащий на полке в вагоне идущего поезда, отноосительно вагона покоится, а относительно Земли движется.
Числа, с помощью которых задается положение точки называются координатами.
Сисиема координат - совокупность условий, определяющих положение точки на прямой, на плоскости, в пространстве.
Тело, относительно которого рассматривается движение называется тело отсчета.
Тело отсчета, система координат, связанная с ним, и прибор для измерения времени называется системой отсчета.
Перемещением называется вектор, проведенный из начального положения тела в конечное.
Траектория - линия, вдоль которой движется тело.
Проводя мелом по доске, мы оставляем на ней след - траекторию движенмя кончика мелка.
Длинна траектории называется путем.
Поступательное движение - движение тела, при котором все его точки двигались одинаково.
Поступательным движение называется такое движение, при котором любая прямая, проведенное в теле.
Вращательным называется такое движение тела, при котором все его точки движутся по окружностям с центрами, лежащими на одной прямой называемой осью вращения.
Материальная точка - это тело, размеры которого можно принебречь в данной задаче.

                         Равномерное прямолинейное движение.
Равномерное движение - движение, при котором тело проходит за равные промежутки времени одинаковые отрезки пути.
Прямолинейное движение - движение тела по прямой.
Равномерное прямолинейное движение(РПД) - это движение тела, при котором за любые равные промежутки времени совершает раввные перемещения.
Скорость РПД - физ. величина равная отношения перемещения к времени.

                         Решение задач на РПД.

                         Средняя скорость. Средняя путевая скорость.
Средней скоростью на данном участке траектории называется физ. величина равная отношению перемещения тела на этом участке ко времени, за которое оно произошло.
Средняя путевая скорость - это отношение длины пути, пройденного телом, ко времени, за которое этот путь был пройден.

                         Решение задач на Среднюю скорость.

                         Мгновеная скорость. Равноускоренное движение. Ускорение.
Мгновенная скорость – скорость движения тела в данный момент времени, скорость тела в данной точке траектории.
Мгновенная скорость всегда направлена по касательной.
Равноускоренным называется такое движение тела, при котором скорость изменяется на одинаковую векторную величину(РУД).
Ускорением тела, при РУД равно отношению изменения скорости ко времени, за которое оно прошло.

                         Решение задач на РУД.

                         Свободное падение. Ускорение свободного падения.
Движение тела под действием только силы тяжести называется свободным падением.
